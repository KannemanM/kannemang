
# Kannemang

**Kannemang** es una galería de imágenes interactiva y responsive basada en la plantilla [Lens](https://html5up.net/lens) de HTML5 UP.

---

## 🚀 Características
- Diseño a pantalla completa  
- Navegación por imágenes usando teclado  
- Adaptable a dispositivos móviles (responsive)  
- Fácil de personalizar y extender  

---

## 📂 Estructura del proyecto
```
|-- assets/           # Archivos CSS y JS
|-- images/           # Imágenes de la galería
|-- index.html        # Página principal
|-- LICENSE.txt       # Licencia
```

---

## ⚡ Instalación
1. Clona el repositorio:  
   ```bash
   git clone https://github.com/KannemanM/kannemang.git
   ```
2. Abre el archivo `index.html` en tu navegador.

---

## 🎮 Uso
- Haz clic en una miniatura para ver la imagen a pantalla completa.  
- Utiliza las flechas del teclado para navegar.

---

## 🛠️ Personalización
1. Agrega nuevas imágenes en la carpeta `images/`.  
2. Edita `index.html` y agrega artículos como:  
   ```html
   <article>
       <a class="thumbnail" href="images/fullsize.jpg">
           <img src="images/thumbnail.jpg" alt="Descripción" />
       </a>
       <h2>Título</h2>
       <p>Descripción breve.</p>
   </article>
   ```
